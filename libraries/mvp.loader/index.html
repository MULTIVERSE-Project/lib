Parameters

  • path

    Path to the file to load.

mvp.loader.LoadFile(path)

 View source »

Send and loads files. This function will determine if the file is a client file, server file or shared file and call the appropriate function.

Parameters

  • path string

    Path to the file to load.

mvp.loader.LoadFolder(path, fromLua)

 View source »

Loads all files in a directory.

Parameters

  • path string

    Path to the directory to load.

  • fromLua boolean default: false

    If true, the path will be relative to the lua/ folder.
